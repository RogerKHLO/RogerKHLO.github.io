The Brief

Critical infrastructure demands absolute data integrity. Standard MQTT implementations often transmit in plain text, leaving systems vulnerable to packet sniffing and Man-in-the-Middle attacks.

The objective was to engineer a hardened IoT solution that enforces a "Zero Trust" model. The system required "military-grade" encryption, strict role-based access control, and automated defense logic to function securely in a hostile network environment.

The Arsenal

The architecture separates concerns between a hardened FreeBSD broker and a dual-core edge device, linked by a custom Public Key Infrastructure.

FreeBSD & Mosquitto
The Broker. A secure OS hosting the MQTT broker, configured to reject all non-encrypted and anonymous connections.
ESP32 (Dual-Core)
The Edge. Custom firmware utilizing FreeRTOS to dedicate Core 0 to encryption/WiFi and Core 1 to sensor logic.
Mutual TLS (PKI)
The Shield. A 3-tier certificate hierarchy (Root, Intermediate, End-Entity) ensuring only verified devices can connect.
Python & Flask
The Console. A retro-terminal dashboard that visualizes telemetry and allows manual command overrides.
SQLite3
The Ledger. Provides forensic auditability by logging every packet topic, payload, and timestamp to a local database.
Access Control Lists
The Gatekeeper. Enforces "Least Privilege," preventing sensors from issuing commands or accessing unauthorized topics.

The Execution

Unlike standard Arduino loops which block network traffic during sensor reads, I implemented a Dual-Core Architecture. Core 0 handles the heavy SSL/TLS handshake calculations, while Core 1 manages the LD2410 Radar and DHT11 sensors, communicating via FreeRTOS queues.

On the server side, Mutual Authentication (mTLS) was enforced. Even if a hacker intercepts the WiFi packets, the payload remains unreadable garbage data due to TLS 1.3 encryption.

The Result

The project successfully demonstrated a system where even a "rogue" device with a valid SSL certificate is rejected by the Access Control List. Packet capture analysis confirmed total obfuscation of application layer data.
